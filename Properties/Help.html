Аккаунты

Root

Root - аккаунт, имеющий полный доступ ко всем данным программы, включая пароли других аккаунтов и их редактирование.
Root создается единожды при первом запуске программы

Администратор

Администратор - аккаунт, имеющий полный доступ ко всем данным программы за исключением данных других аккаунтов.
Администратор может создавать, удалять и просматривать данные (за исключением паролей) аккаунты только для обычных пользователей.

Пользователь

Пользователь - аккаунт имеющий доступ на просмотр всех данных программы за исключением данных других аккаунтов.
Пользователь может просматривать и редактировать только свой аккаунт.

Работа с данными

Для работы с данными необходимо перейти на соответствующую форму, переход к которой осуществляется в левой части главного окна

Студенты

Данные о студентах содержат следующие поля:

  • Фамилия
  • Имя
  • Отчество
  • Фамилия
  • Форма обучения
  • Активное участие в общественной деятельности
  • Группа

Данные о студентах могут быть сгруппированы по группам если выбрать группу во всплывающем списке находящимся в верхнем-правом углу

Группы

Данные о группах содержат следующие поля:

  • Номер группы
  • Куратор

Предметы

Данные о предметах содержат следующие поля:

  • Название предмета

Пользователи

Данные о пользователях содержат следующие поля:

  • Логин
  • Пароль
  • Роль

Работа с табелем оценок

Для создания табеля оценок необходимо перейти в пункт меню Файл -> Создать.
После этого откроется панель редактирования табеля оценок.

Для открытия существующего табеля оценок необходимо перейти в пункт меню Файл -> Открыть и выбрать файл.

Для сохранения табеля оценок необходимо перейти в пункт меню Файл -> Сохранить(Сохранить как - для создания нового фала) и выбрать путь сохранения.

Для добавления студента в табель нажмите кнопку "Добавить студента" находящуюся в нижней части панели редактирования табеля, после чего всплывет окно с выбором студентов.
Выберите студента (Ctrl или Shift для выбора нескольких студентов) и нажмите кнопку "ОК"

Для добавления предмета в табель нажмите кнопку "Добавить предмет" находящуюся в нижней части панели редактирования табеля, после чего всплывет окно с выбором предмета.
Выберите предмет (Ctrl или Shift для выбора нескольких предметов) и нажмите кнопку "ОК"
